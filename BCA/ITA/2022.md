## Section 1: Short Answer Questions (Answer Any Six)

### (a) What is the Purpose of an Operating System?

The purpose of an operating system is to act as an **interface between the user and the computer hardware**.

It manages all hardware and software resources of the computer. It also controls the execution of programs and makes the computer easy to use.

**Conclusion:** Without an operating system, a computer cannot work properly.

---

### (b) What is the Difference Between Program and Software?

**📘 Program**

**Definition:** A program is a set of instructions written in a programming language that a computer can understand and execute. It tells the computer step-by-step what task to perform.

**💻 Software**

**Definition:** Software is a set of programs and instructions that tells a computer what to do. It is the intangible part of a computer system, which means it cannot be seen or touched. Software helps the computer to perform different tasks.

**Example:**
- Program → Calculator program
- Software → Microsoft Office

**Conclusion:** Program is a part of software, while software is a complete package.

---

### (c) What is Warm Booting?

**Warm booting** means restarting the computer without switching off the power.

It is done by using restart option or pressing **Ctrl + Alt + Delete**.

Warm booting is faster than cold booting.

**Conclusion:** It helps to restart the system quickly without turning off the computer.

---

### (d) What is the Difference Between Data and Information?

**Data:** Raw facts and figures which do not have any meaning.

**Information:** Processed data which is meaningful and useful.

**Example:**
- Marks = Data
- Result = Information

**Conclusion:** Data becomes information after processing.

---

### (e) What is the Purpose of Transmission Media?

Transmission media is used to transfer data from one device to another in a network.

It provides a path for communication between computers.

**Conclusion:** It is necessary for data communication.

---

### (f) What is the Basic Need of Hypertext?

Hypertext is used to link one document to another document.

It helps users to move easily from one page to another on the internet.

**Conclusion:** It makes navigation easy and fast.

---

### (g) What is Database?

Optical disk is used to store data using **laser light technology**.

### Question 3: Internal Commands vs External Commands in DOS

#### Introduction

DOS (Disk Operating System) provides various commands to perform different operations like file handling, directory management, etc. These commands are mainly divided into two types: **Internal Commands** and **External Commands**.

---

#### Internal Commands

**Definition:** Internal commands are those commands which are stored in the main memory and are always available when the computer is running.

**Features:**
- Stored in memory
- Executed quickly
- Always available

**Examples with Syntax:**

1. **DIR (Directory)**
   - Used to display list of files and folders
   - **Syntax:** `DIR`

2. **COPY**
   - Used to copy one file to another
   - **Syntax:** `COPY file1 file2`

3. **DEL (Delete)**
   - Used to delete a file
   - **Syntax:** `DEL filename`

---

#### External Commands

**Definition:** External commands are those commands which are stored on disk as separate files and are loaded into memory only when needed.

**Features:**
- Stored on disk
- Not always available
- Slower than internal commands

**Examples with Syntax:**

1. **FORMAT**
   - Used to format a disk
   - **Syntax:** `FORMAT C:`

2. **CHKDSK**
   - Used to check the disk for errors
   - **Syntax:** `CHKDSK`

3. **XCOPY**
   - Used to copy files and directories
   - **Syntax:** `XCOPY source destination`

---

#### Conclusion

Both internal and external commands are important in DOS. Internal commands are used for basic operations, while external commands are used for advanced operations. Together, they help in managing files and system functions effectively.

### Question 4: Components of Computer with Block Diagram

#### Introduction

A computer system is made up of different components which work together to perform tasks. These components include **Input Unit**, **Central Processing Unit (CPU)**, **Memory Unit**, and **Output Unit**. All these parts are connected and work step by step to process data.

---

#### Block Diagram

```
┌─────────────┐
│ Input Unit  │
└──────┬──────┘
       │
       ▼
┌─────────────┐      ┌──────────────┐
│     CPU     │◄────►│ Memory Unit  │
└──────┬──────┘      └──────────────┘
       │
       ▼
┌─────────────┐
│ Output Unit │
└─────────────┘
```

---

#### 🖥️ Categories of Computer Hardware

Computer hardware is divided into **four main categories**:

##### 1. Input Devices

**Definition:** Input devices are used to enter data and instructions into the computer.

**Examples:**
- **Keyboard** (for typing)
- **Mouse** (for pointing and clicking)
- **Scanner** (for scanning documents)

---

##### 2. Output Devices

**Definition:** Output devices are used to display or show the result processed by the computer.

**Examples:**
- **Monitor** (shows output on screen)
- **Printer** (prints on paper)
- **Speakers** (produce sound)

---

##### 3. Processing Unit (CPU)

**Definition:** CPU (Central Processing Unit) is the brain of the computer. It processes data and controls all operations of the computer.

**CPU has two main parts:**

**(a) Arithmetic Logic Unit (ALU):**
- Performs calculations like addition, subtraction
- Performs logical operations

**(b) Control Unit (CU):**
- Controls all activities of computer
- Directs data flow between components

**Working:** CPU receives data from input unit, processes it, and sends result to output unit.

---

##### 4. Storage Devices

**Definition:** Storage devices are used to store data and information for future use.

**Examples:**
- Hard Disk
- SSD
- USB Drive

---

#### Working of Computer System

1. User gives input
2. Input unit sends data to CPU
3. CPU processes data using memory
4. Output unit displays result

---

#### Conclusion

All components of a computer work together in a proper sequence. Input unit gives data, CPU processes it, memory stores it, and output unit shows the result. This complete process makes the computer useful and efficient.

### Question 5: Computer Output Microfilm & Secondary Storage Devices

#### Introduction

Computer Output Microfilm (COM) and secondary storage devices are important concepts in computer systems. COM is used for storing output in microfilm form, while secondary storage devices are used for permanent data storage.

---

#### 📘 Computer Output Microfilm (COM)

**🔹 Definition**

Computer Output Microfilm (COM) is a technology used to store computer output on microfilm instead of paper. It converts digital data into very small images on film.

**🔹 What is Microfilm?**

Microfilm is a thin film that stores images in very small size. Large data can be stored in a compact form.

**Explanation:**

In COM, data from the computer is recorded on a microfilm using special equipment. It is mainly used to store large amounts of information in a very small space.

**Advantages:**
- Saves storage space
- Long-term data storage
- Cost-effective for large data

**Uses:**
- Libraries
- Offices
- Record keeping systems

---

#### 📘 Classification of Secondary Storage Devices

**🔹 Secondary Storage Devices**

Secondary storage devices are used to store large amounts of data permanently. They are non-volatile, which means the data is not lost when the power is turned off. These devices are used for long-term storage of data and programs.

---

##### 📼 Magnetic Storage Devices

These devices store data using magnetic fields.

**Examples:**
- **Hard Disk Drive (HDD)** - A type of magnetic disk that stores large amounts of data inside a computer
- **Magnetic Tape** - Stores data on a long plastic tape coated with magnetic material

**Features:**
- Large storage capacity
- Used for long-term storage
- Data access is sequential in tapes

---

##### 💿 Optical Storage Devices

These devices use laser light to read and write data.

**Examples:**
- **CD (Compact Disc)** - Can store up to 700 MB of data
- **DVD (Digital Versatile Disc)** - Can store between 4.7 GB to 17 GB
- **Blu-ray Disc** - Uses blue laser technology for high-definition data storage

**Features:**
- Portable
- Used for multimedia storage
- Good storage capacity

---

##### 💡 Solid State Storage Devices

These devices use electronic circuits to store data.

**Examples:**
- **Pen Drive** (USB Drive)
- **SSD (Solid State Drive)**
- **Memory Cards**

**Features:**
- Fast speed
- No moving parts
- Reliable

---

##### 💡 Magneto-Optical Disk

A magneto-optical disk is a storage device that combines both magnetic and optical technologies. It uses heat and magnetic fields to write data and laser to read it.

**Features:**
- Rewritable
- Long data life
- Expensive and slower compared to modern devices

---

#### Conclusion

COM is useful for storing large data in microfilm form, while secondary storage devices are used for permanent data storage. Different types of storage devices help in storing and managing data efficiently.

### Question 7: Layers of TCP/IP Model

#### Introduction

The **TCP/IP model** is a set of rules used for communication over the internet. It helps different computers to communicate with each other in a network.

**TCP/IP** stands for **Transmission Control Protocol / Internet Protocol**.

This model is divided into **four layers**, and each layer performs a specific function.

---

#### Layers of TCP/IP Model

##### 1. Application Layer

This is the **top layer** of the TCP/IP model. It provides services directly to the user.

**Functions:**
- Allows users to access network services
- Supports applications like web browsers and email

**Examples:** HTTP, FTP, SMTP

---

##### 2. Transport Layer

This layer is responsible for delivering data from one device to another.

**Functions:**
- Ensures reliable data transmission
- Divides data into small parts (segments)
- Provides error checking

**Protocols:** TCP, UDP

---

##### 3. Internet Layer

This layer handles addressing and routing of data.

**Functions:**
- Assigns IP address
- Sends data packets from source to destination

**Protocol:** IP (Internet Protocol)

---

##### 4. Network Access Layer

This is the **lowest layer** of the TCP/IP model.

**Functions:**
- Handles physical transmission of data
- Controls hardware devices

**Examples:** Ethernet, Wi-Fi

---

#### Working of TCP/IP Model

1. Data starts from **Application Layer**
2. Moves to **Transport Layer**
3. Then to **Internet Layer**
4. Finally to **Network Access Layer**
5. At receiver side, process happens in **reverse order**

---

#### Conclusion

TCP/IP model is very important for internet communication. Each layer has its own function, and together they ensure smooth and reliable data transfer between computers.
