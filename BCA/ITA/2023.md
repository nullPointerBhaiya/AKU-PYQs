### Question 3: Different Types of Switching Technique

**Answer:**

Switching technique is used in computer networks to send data from one device to another. There are mainly three types of switching: circuit switching, packet switching, and message switching. In circuit switching, a direct path is created between sender and receiver before communication starts, like in telephone calls. In packet switching, data is divided into small packets and sent separately through different paths, which is used in the internet. In message switching, the whole message is sent at once and stored before forwarding. Packet switching is most commonly used today because it is faster and more efficient than other methods.

### Question 5: Computer Output Microfilm & Secondary Storage Devices

#### Computer Output Microfilm (COM)

**Definition:** Computer Output Microfilm (COM) is a technology used to store computer output on microfilm instead of paper. It converts digital data into very small images on film.

**What is Microfilm?** Microfilm is a thin film that stores images in very small size. Large data can be stored in a compact form.

**Explanation:** In COM, data from the computer is recorded on a microfilm using special equipment. It is mainly used to store large amounts of information in a very small space.

**Advantages:**
- Saves storage space
- Long-term data storage
- Cost-effective for large data

**Uses:**
- Libraries
- Offices
- Record keeping systems

---

#### 📘 Classification of Secondary Storage Devices

**Secondary storage devices** are used to store large amounts of data permanently. They are non-volatile, which means the data is not lost when the power is turned off. These devices are used for long-term storage of data and programs.

---

##### 📼 Magnetic Storage Devices

These devices store data using magnetic fields.

**Examples:**
- **Hard Disk Drive (HDD)** - A type of magnetic disk that stores large amounts of data inside a computer
- **Magnetic Tape** - Stores data on a long plastic tape coated with magnetic material

**Features:**
- Large storage capacity
- Used for long-term storage
- Data access is sequential in tapes

---

##### 💿 Optical Storage Devices

These devices use laser technology to read and write data.

**Examples:**
- **CD (Compact Disc)** - Can store up to 700 MB of data
- **DVD (Digital Versatile Disc)** - Can store between 4.7 GB to 17 GB
- **Blu-ray Disc** - Uses blue laser technology for high-definition data storage

**Features:**
- Portable
- Used for multimedia storage
- Good storage capacity

---

##### 💡 Solid State Storage Devices

These devices use electronic circuits to store data.

**Examples:**
- **Pen Drive** (USB Drive)
- **SSD (Solid State Drive)**
- **Memory Cards**

**Features:**
- Fast speed
- No moving parts
- Reliable

---

##### 💡 Magneto-Optical Disk

A magneto-optical disk is a storage device that combines both magnetic and optical technologies. It uses heat and magnetic fields to write data and laser to read it.

**Features:**
- Rewritable
- Long data life
- Expensive and slower compared to modern devices

---

#### Conclusion

COM is useful for storing large data in microfilm form, while secondary storage devices are used for permanent data storage. Different types of storage devices help in storing and managing data efficiently.

#### 📘 Algorithm to Calculate Factorial of N

**Definition:** An algorithm is a step-by-step procedure used to solve a problem. It shows the sequence of steps that must be followed to get the correct result in a clear and simple way.

---

**Algorithm to Calculate Factorial of N:**

```
Step 1: Start
Step 2: Input number N
Step 3: Set FACT = 1
Step 4: Set I = N
Step 5: Repeat while I >= 1
        FACT = FACT * I
        I = I - 1
Step 6: Display FACT
Step 7: Stop
```

**Explanation:**

To calculate the factorial of a number N, we follow these steps: start the process, take input N, set a variable fact to 1, then multiply fact by all numbers from N to 1, and finally display the result and stop. This method helps to find the product of all numbers from 1 to N.

**Example:** Factorial of 5 = 5 × 4 × 3 × 2 × 1 = 120

### Question 7: Layers of TCP/IP Model

#### Introduction

The **TCP/IP model** is a set of rules used for communication over the internet. It helps different computers to communicate with each other in a network.

**TCP/IP** stands for **Transmission Control Protocol / Internet Protocol**.

This model is divided into **four layers**, and each layer performs a specific function.

---

#### Layers of TCP/IP Model

##### 1. Application Layer

This is the **top layer** of the TCP/IP model. It provides services directly to the user.

**Functions:**
- Allows users to access network services
- Supports applications like web browsers and email

**Examples:** HTTP, FTP, SMTP

---

##### 2. Transport Layer

This layer is responsible for delivering data from one device to another.

**Functions:**
- Ensures reliable data transmission
- Divides data into small parts (segments)
- Provides error checking

**Protocols:** TCP, UDP

---

##### 3. Internet Layer

This layer handles addressing and routing of data.

**Functions:**
- Assigns IP address
- Sends data packets from source to destination

**Protocol:** IP (Internet Protocol)

---

##### 4. Network Access Layer

This is the **lowest layer** of the TCP/IP model.

**Functions:**
- Handles physical transmission of data
- Controls hardware devices

**Examples:** Ethernet, Wi-Fi

---

#### Working of TCP/IP Model

1. Data starts from **Application Layer**
2. Moves to **Transport Layer**
3. Then to **Internet Layer**
4. Finally to **Network Access Layer**
5. At receiver side, process happens in **reverse order**

---

#### Conclusion

The TCP/IP model is used for communication over the internet. It has four layers: Application layer, Transport layer, Internet layer, and Network Access layer. The Application layer provides services like email and web browsing to users. The Transport layer ensures that data is delivered correctly using protocols like TCP and UDP. The Internet layer handles addressing and routing of data using IP addresses. The Network Access layer manages the physical transmission of data through cables or wireless signals. Each layer has its own function, and together they help in sending and receiving data smoothly across networks.

### (b) Draw a Flowchart to Find the Largest Among Three Numbers

**Answer:**

To find the largest among three numbers using a flowchart, we start the process and take three inputs A, B, and C. First, we compare A with B and C. If A is greater than both, then A is the largest number. If not, we compare B with C. If B is greater, then B is the largest. Otherwise, C is the largest number. After finding the largest value, we display the result and stop the process. This flowchart helps in solving the problem step by step in a clear and simple way, making it easy to understand the logic.

**Flowchart Structure:**

```
START
  ↓
INPUT A, B, C
  ↓
IS A > B AND A > C?
  ↓ YES          ↓ NO
LARGEST = A    IS B > C?
  ↓              ↓ YES      ↓ NO
DISPLAY        LARGEST = B  LARGEST = C
LARGEST          ↓            ↓
  ↓            DISPLAY      DISPLAY
STOP           LARGEST      LARGEST
                 ↓            ↓
               STOP         STOP
```

**Answer:**

RAM and ROM are two types of memory used in a computer.

**RAM (Random Access Memory):**
- RAM stands for Random Access Memory and it is temporary memory
- It stores data and programs while the computer is working
- When the power is turned off, all data in RAM is lost
- RAM is faster and used for current work

**ROM (Read Only Memory):**
- ROM stands for Read Only Memory and it is permanent memory
- It stores important instructions that help the computer to start

### (e) Explain the Idea of Multiplexing

**Answer:**

Multiplexing is a technique used in communication systems to send multiple signals through a single channel. It helps in better use of available resources and reduces the cost of transmission. In this method, different signals are combined at the sender side and then separated again at the receiver side. This allows many users to share the same communication line at the same time. Multiplexing is widely used in telephone networks, internet communication, and data transmission systems. It improves efficiency and makes communication faster and more organized.
