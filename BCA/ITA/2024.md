### (a) Semiconductor Memory Device

**Question:** Semiconductor memory device in which stored data will remain permanently stored as long as power is supplied is:

- (i) Dynamic memory device
- (ii) Volatile memory device

**⚡ SRAM (Static RAM)** is a type of RAM that stores data using flip-flop circuits. It is very fast and does not need to be refreshed again and again. It is expensive and mainly used in cache memory. As long as power is supplied, data remains stored.

### Question 3: Basic Characteristics of Computer

#### 📘 Characteristics of Computer (SAVAS DMC)

A computer has many important characteristics that make it very useful in daily life.

---

#### ⚡ Speed

Speed means a computer can work very fast. It can perform millions of calculations in one second. This helps users complete tasks quickly and saves time. Speed is one of the most important features of a computer system.

---

#### 🎯 Accuracy

Accuracy means a computer gives correct results. If the input and program are correct, the output will also be correct. Computers do not make mistakes on their own. Errors happen only due to wrong data or instructions given by humans.

---

#### 🤖 Automation

Automation means a computer works automatically after instructions are given. Once a program starts, the computer completes the task without human help. This reduces manual work and increases efficiency in performing repeated tasks easily and quickly.

---

#### 💾 Storage

Storage means a computer can store large amounts of data. It can save data permanently or temporarily for future use. Users can easily access stored data anytime. Devices like hard disk, SSD, and USB are used for storage.

---

#### 🔄 Versatility

Versatility means a computer can perform many different types of tasks. It can be used in education, business, entertainment, and more. A single computer can do calculations, typing, designing, and internet browsing at the same time.

---

#### 🧘 Diligence

Diligence means a computer does not get tired or bored. It can work continuously for a long time without losing accuracy or speed. Unlike humans, it performs repetitive tasks again and again with the same efficiency.

---

#### 🔀 Multitasking

Multitasking means a computer can perform multiple tasks at the same time. For example, you can listen to music, browse the internet, and type a document together. This feature increases productivity and saves time for users.

---

#### 🌐 Communication

Communication means a computer can connect with other devices through networks or the internet. It allows sharing of data and information quickly. Computers are used for email, video calls, and online communication across the world.

---

#### Conclusion

These features make computers very useful in daily life. A computer is fast, accurate, reliable, and versatile, which is why it is used in almost every field today.

### Question 4: Classification of Storage Devices

#### Introduction

Storage devices are used to store data in a computer. They are mainly classified into **primary storage** and **secondary storage**. Both types are important for proper functioning of a computer.

---

#### 📘 Memory System

**🔹 Memory System**

A memory system is the part of a computer that is used to store data, instructions, and results. It helps the computer to remember information temporarily or permanently so that it can be used when needed.

---

#### Primary Storage (Main Memory)

Primary storage includes RAM and ROM, which are used during processing.

##### 💾 RAM (Random Access Memory)

RAM is a type of primary memory that is used to store data and programs currently being used by the computer. It is temporary in nature, which means all data is lost when the power is turned off.

**Types of RAM:**

**⚡ SRAM (Static RAM)**
- Stores data using flip-flop circuits
- Very fast and does not need to be refreshed
- Expensive and mainly used in cache memory

**⚡ DRAM (Dynamic RAM)**
- Stores data using capacitors
- Needs to be refreshed continuously
- Slower than SRAM but cheaper
- Used as the main memory in computers

---

##### 💽 ROM (Read Only Memory)

ROM is a type of memory that stores permanent data and instructions. It is non-volatile, which means it keeps data even when the power is turned off. It is mainly used to store system programs like BIOS.

**Types of ROM:**

**🔸 Mask ROM** - Programmed during manufacturing, cannot be changed

**🔸 PROM (Programmable ROM)** - Can be programmed once by the user

**🔸 EPROM (Erasable Programmable ROM)** - Can be erased using ultraviolet light and reprogrammed

**🔸 EEPROM (Electrically Erasable Programmable ROM)** - Can be erased and reprogrammed using electrical signals

**🔸 Flash Memory** - Advanced type of EEPROM, used in USB drives, memory cards, and SSDs

---

#### Secondary Storage (Auxiliary Storage)

Secondary storage includes hard disk, CD, DVD, and pen drive, which store data permanently.

Secondary storage is further divided into:

##### 📼 Magnetic Storage

**Magnetic Tape** - Stores data on a long plastic tape coated with magnetic material. Data is stored sequentially. High storage capacity and low cost, but data access is slow. Mainly used for backup and long-term storage.

**💽 Magnetic Disk** - Stores data on rotating disks with magnetic surfaces. Allows random access. High capacity and moderate speed.

**🔹 Hard Disk Drive (HDD)** - A type of magnetic disk that stores large amounts of data inside a computer. Uses rotating platters and read/write heads.

---

##### 💿 Optical Storage

**Optical Disk** - Uses laser technology to read and write data. Stores data in the form of tiny pits on the surface.

**🔹 CD (Compact Disc)** - Can store up to 700 MB of data. Mainly used for music, software, and data storage.

**🔹 DVD (Digital Versatile Disc)** - Can store between 4.7 GB to 17 GB. Commonly used for movies and large data storage.

**🔹 Blu-ray Disc** - Uses blue laser technology to store high-definition data. Very high storage capacity, used for HD videos and large files.

---

##### 💡 Solid-State Storage

**🔸 Flash Memory** - Fast, reliable, and widely used in devices like USB drives, memory cards, and SSDs.

**Examples:**
- Pen Drive (USB Drive)
- SSD (Solid State Drive)
- Memory Cards

**Features:**
- Fast speed
- No moving parts
- Reliable

---

##### 💡 Magneto-Optical Disk

Combines both magnetic and optical technologies. Uses heat and magnetic fields to write data and laser to read it. Rewritable and has long data life but is expensive and slower compared to modern devices.

---

#### Conclusion

Primary memory is fast but temporary, while secondary memory is slower but permanent. Both types are important for proper functioning of a computer.

### Question 5: Characteristics of Good Programming Language

#### 📘 Features of a Good Programming Language (SUPREME S)

#### 🔹 Introduction

A good programming language should be simple and easy to understand. It should be easy to learn and use. It must be reliable and give correct results. These features help programmers to write better and error-free programs.

---

---

---

---

Modularity means the program can be divided into smaller parts or modules. These modules can be reused, making programming easier and more organized.

---

### Question 6: Functions of Operating System

#### 📘 Operating System (OS)

**🔹 Definition**

An Operating System is system software that acts as an interface between user and computer hardware. It controls all activities of the computer and helps users to run programs easily.

---

#### ⚙️ Functions of Operating System

An Operating System manages all activities of a computer. Without an operating system, a computer cannot work properly. It is one of the most important parts of a computer system.

---

##### 🔹 Process Management

Process management means the operating system controls and manages running programs. It decides which process will run first and how much time it will get. It also handles multitasking (running multiple programs at the same time).

---

##### 🔹 Memory Management

Memory management means the OS allocates and frees memory as needed. It ensures proper use of memory by different programs. The OS manages the main memory (RAM) and allocates memory to different programs and removes it when not needed.

---

##### 🔹 File Management

File management means the OS organizes and controls files and folders. It helps users to store, access, and manage data easily. It helps in creating, deleting, reading, and writing files.

---

##### 🔹 Device Management

Device management means the OS controls input and output devices like keyboard, mouse, and printer. It uses drivers to communicate with hardware. The OS controls all hardware devices and uses device drivers to communicate with hardware.

---

##### 🔹 Security

Security means the OS protects the system from unauthorized access. It ensures data safety using passwords and permissions. The OS protects data from unauthorized access and provides features like passwords and user authentication.

---

##### 🔹 User Interface

User interface allows users to interact with the computer. It can be CLI (command line) or GUI (graphical interface). The OS provides an interface for users to interact with the computer.

**Two types:**
- Command Line Interface (CLI)
- Graphical User Interface (GUI)

---

##### 🔹 Input and Output Management

The OS manages input and output operations. It controls how data is received from input devices and sent to output devices.

---

##### 🔹 Error Handling

The OS detects errors in hardware and software and takes necessary actions. It helps in maintaining system stability.

---

##### 🔹 Resource Management

The OS manages all resources like CPU, memory, and devices. It ensures proper and fair use of resources.

---

#### Conclusion

The OS acts as a link between user and hardware. It allows users to run programs easily. Operating system plays a very important role in the functioning of a computer. It manages hardware, software, and provides a user-friendly environment. Without OS, a computer system cannot work efficiently.

### Question 7: Transmission Media and Its Types

#### Introduction

Transmission media is the path through which data travels from one device to another in a network. It is used for communication between computers. It plays an important role in networking and communication systems.

---

#### What is Transmission Media?

Transmission media is the medium or path through which data is transmitted from one device to another in a network. It helps in sending data quickly and correctly.

---

#### Types of Transmission Media

There are **two main types** of transmission media: **guided** and **unguided**.

---

#### 1. Guided Media (Wired Media)

Guided media uses physical cables to transmit data. In this type, signals travel through wires. It is more secure and less affected by noise.

**Examples of guided media:**

##### Twisted Pair Cable
- Most common type of cable
- Used in telephone lines and LANs
- Inexpensive and easy to install

##### Coaxial Cable
- Better quality than twisted pair
- Used in cable TV and internet
- Less interference

##### Optical Fibre Cable
- Uses light signals for communication
- Very fast and high bandwidth
- Long distance communication
- Less signal loss
- Most expensive but most efficient

---

#### 2. Unguided Media (Wireless Media)

Unguided media does not use any physical cables. Data is transmitted through air using electromagnetic waves.

**Examples:**

##### Radio Waves
- Used for long distance communication
- Can penetrate walls
- Used in radio broadcasting and Wi-Fi

##### Microwaves
- Used for point-to-point communication
- Requires line of sight
- Used in satellite communication

##### Satellite Communication
- Used for very long distance communication
- Covers large geographical areas
- Used in GPS and weather forecasting

---

#### Advantages and Disadvantages

**Guided Media:**
- ✓ More secure
- ✓ Less interference
- ✗ Installation cost is high
- ✗ Limited mobility

**Unguided Media:**
- ✓ Easy to install
- ✓ Useful for long-distance communication
- ✓ Mobile connectivity
- ✗ Less secure
- ✗ More interference

---

#### Conclusion

Both guided and unguided media are important for data transmission depending on requirement. Transmission media helps in sending data quickly and correctly between devices in a network.
